# DynamicIslandLyrics 🎵

一个使用 **C# / WPF** 编写的桌面歌词悬浮窗，外观参考苹果灵动岛（Dynamic Island）。

**完全由 AI VibeCoding 生成**，全程通过 [AtomCode](https://atomgit.com/)（Claude Code 的 AtomGit 定制版）在自然语言驱动下编写。未手动写过一行代码。

## 功能

- 🏝️ **灵动岛风格** — 无边框半透明窗口，底部圆角 42px
- 📡 **SMTC 集成** — 读取任意支持 SMTC 的播放器的曲名、歌手、进度、封面
- 🎤 **智能歌词查找** — 多层回退链保证歌词命中率
- 🎞️ **歌词动画** — 淡入淡出 / 缩放进入，通过系统托盘右键菜单切换
- 📌 **始终置顶** — 双击窗口或托盘菜单切换
- 🎯 **贴顶自适应** — 窗口拖到屏幕顶部时自动消除上边距 + 移除上圆角
- 🚫 **Snap 拦截** — 阻止 Win11 分屏覆盖层弹出
- 🔔 **系统托盘** — 右键菜单管理所有设置，双击显示窗口
- 📝 **本地 LRC** — 支持 `Lyrics/` 目录的 `.lrc` 歌词文件

## 歌词来源优先级

```
AMLL TTML 歌词站 → QQ 音乐 → 网易云音乐 → LRCLIB → 本地 LRC 文件
```

| 来源 | 接口 |
|------|------|
| [AMLL TTML 歌词站](https://amlldb.bikonoo.com) | Apple Music Like Lyrics 风格 TTML 歌词 |
| QQ 音乐 | `u.y.qq.com` 搜索 + `c.y.qq.com` 歌词获取 |
| 网易云音乐 | WeApi 加密接口 |
| [LRCLIB](https://lrclib.net) | 开源歌词数据库 |
| 本地文件 | `Lyrics/` 目录下的 `.lrc` 文件 |

## 本地歌词

将 `.lrc` 文件放入 `Lyrics/` 目录，文件名支持以下格式：

- `歌手 - 歌名.lrc`
- `歌名 - 歌手.lrc`
- `歌名.lrc`

```
[00:01.20]第一句歌词
[00:04.80]第二句歌词
```

## 快速开始

### 直接运行（推荐）

从 [Releases](../../releases) 下载 `DynamicIslandLyrics.zip`，解压后双击 `DynamicIslandLyrics.exe` 即可。

> 不需要安装 .NET SDK。单文件自包含，~78 MB。

### 从源码构建

```powershell
# 需要 .NET 8 SDK
dotnet run
```

### 打包分发

```powershell
.\publish.bat
# 输出在 bin\Release\net8.0-windows10.0.19041.0\win-x64\publish\
```

## 操作说明

| 操作 | 效果 |

## 系统托盘

窗口默认不显示在任务栏（`ShowInTaskbar=False`），通过系统托盘图标管理：

- **左键双击** — 显示/激活窗口
- **右键** — 弹出设置菜单
- 窗口关闭后仍在托盘运行，右键菜单选择"退出"才会完全退出

## 日志

运行目录会生成 `smtc-sync.log`，用于排查 SMTC 读取到的曲名、歌手、时长、位置和歌词来源。
